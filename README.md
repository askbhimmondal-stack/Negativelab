Negative Lab is a lightweight and easy-to-use web tool that allows users to convert any image into its negative (color-inverted) version instantly. The tool works directly in the browser and uses the HTML5 Canvas API to process image pixels and generate the negative effect in real time.

Negative Lab is designed for designers, photographers, developers, and students who want to experiment with image colors, create artistic negative effects, or study image processing techniques.

The entire conversion process happens locally in the browser, which means your images are never uploaded to any server, ensuring complete privacy and fast performance.

🛠️ Tech Stack

Negative Lab is built using simple and modern web technologies:

HTML5 – Page structure
CSS3 – Styling and layout
JavaScript – Image processing logic
Canvas API – Pixel manipulation and color inversion
⚙️ How Negative Lab Works
Upload or drag an image into the tool.
The image is loaded into the browser using the Canvas API.
The tool reads every pixel’s RGB value.
Each color value is inverted to produce the negative effect.
The negative image is displayed instantly.
Users can download the processed image.

Example color transformation:

Original RGB → (255, 200, 100)
Negative RGB → (0, 55, 155)
